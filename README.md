# SharingNE

Scambio file diretto tra i tuoi dispositivi sulla **stessa Wi-Fi**, peer-to-peer (WebRTC).
Niente cloud, niente server intermedio: dopo un aggancio iniziale via QR, i file passano
direttamente da un dispositivo all'altro.

Tutti i file stanno nella radice del repository (struttura piatta): l'app li carica da lì.
